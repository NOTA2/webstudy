# webstudy

## 기본사항
- 시간 : (1/3~1/17) 매주 화, 목 오후 2시
- 장소 : G103

## 스터디 전 준비사항
### 팀장 (전현태)
- 진도 확인
- 출석 확인

### 강의 스터디 인원
- 강의 내용 정하기
- 실습 내용 정하기
- 시험 내용 정하기

### 일반 스터디 인원
- 정해준 진도 다 나가고 진도표에 표시

## 스터디 진행
- 강의 (30~40분 분량)
- 실습 (20~30분 분량)
- 쪽지 시험 (10분)

## 벌금 (1,000원)
- 예기치 못한 상황을 제외한 지각/결석 (수업시작 1시간 전까지만 말하면 OK)
- 정해둔 진도를 다 나가지 못한 경우
	※ 다만 2/3 이상 진행하지 못한 경우 벌금을 걷지 않고 진도내역 수정.
